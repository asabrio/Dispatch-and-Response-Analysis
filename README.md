## Code for:
“Dispatch Delays and Geographic Disparities in Police Response Times in New Orleans”

Authors: Alexandra Sabrio, Debashis Mondal

## Overview

This repository contains the code used in the analyses and figures for the manuscript “Dispatch Delays and Geographic Disparities in Police Response Times in New Orleans.” Scripts are provided to reproduce all results reported in the paper.

## Data Download

The analysis relies on the Calls for Service 2018 dataset from the City of New Orleans.
You may download the data in one of two ways:

**From this repository:** The file Calls_for_Service_2018.xlsx is available for download.

**From the official source:** The dataset can be downloaded from the City of New Orleans open data portal:
[City of New Orleans Open Data Portal](https://data.nola.gov/Public-Safety-and-Preparedness/Calls-for-Service-2018/9san-ivhk/about_data)

To access the R data files containing model output without running the code files, please download the .rds data files from the repository.
